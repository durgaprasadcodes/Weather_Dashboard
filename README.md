# 🌦️ Weather Dashboard

A modern, responsive **Weather Dashboard** web application that allows users to search for real-time weather information of any city worldwide.  
Built using **HTML, CSS, and JavaScript**, this project demonstrates API integration, async programming, and clean UI design.

---

## 🚀 Live Demo
🔗 [Click here to visit](https://durgaprasadcodes.github.io/Weather_Dashboard/)

---

## ✨ Features

- 🔍 Search weather by city name
- 🌡️ Displays temperature, humidity, and weather condition
- ☁️ Dynamic weather descriptions & icons
- ❌ Error handling for invalid city names
- 📱 Fully responsive design
- ⚡ Fast and lightweight

---

## 🛠️ Technologies Used

- **HTML5** – Structure & markup  
- **CSS3** – Styling & responsive layout  
- **JavaScript (ES6)** – Logic & API handling  
- **Weather API** – Real-time weather data

## 🔧 How It Works

1. User enters a **city name**
2. JavaScript sends a request to the **Weather API**
3. API responds with real-time weather data
4. UI updates dynamically with:
   - Temperature
   - Humidity
   - Weather description
5. Errors are shown for invalid inputs
